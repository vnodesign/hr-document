# Ngôn ngữ lập trình JavaScript

Mình thấy không ít người nhầm lẫn giữa hai ngôn ngữ lập trình Javascript và ngôn lập trình Java do tên gọi của chúng khá giống nhau. Trong giới Back End, thì JS khá được ưa chuộng vì là ngôn ngữ bậc cao có tính động, không định kiểu (un-typed) và có khả năng thông dịch.
