**Các cấu hình khác trên Card và Card Group:**

<ResponseField name="title" type="string">
`title` là tiêu đề mà bạn muốn đặt trong `Card`.

Ví dụ: `Front-end là gì`

</ResponseField>

<ResponseField name="icon" default="Folder" type="string">
`icon` là biểu tượng mà bạn muốn đặt trong `Card`.

Ví dụ: `TableLayout`

Danh sách các Icon đang sử dụng trên dự án HR Documentation:

- `ArrowLeft`
- `ArrowLeftShort`
- `ArrowRight`
- `ArrowRightShort`
- `CaretDown`
- `CaretUp`
- `Chat`
- `Check`
- `Facebook`
- `GitHub`
- `Linkedin`
- `ReactIcon`
- `Twitter`
- `TableLayout`
- `Server`
- `SwatchBook`
- `Folder`
- `FileCode`

</ResponseField>

<ResponseField name="href" type="string">
`href` là liên kết mà bạn muốn đặt trong `Card`.

Ví dụ: `https://hrpenci.site/group-hr`

</ResponseField>

<ResponseField name="cols" default="2" type="number">
`cols` là số cột mà bạn muốn đặt trong `CardGroup`.

<Note>Trên các thiết bị di động, thì các cột sẽ hiển thị thành một.</Note>
