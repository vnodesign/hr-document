# API là gì?

![API là gì](/static/docs/api.jpg)

API là viết tắt của cụm từ "Application Programming Interface", đây là một khái
niệm rất phổ biến trong lĩnh vực phát triển phần mềm. API là một tập hợp các
giao diện (interfaces) cho phép các ứng dụng và dịch vụ khác nhau tương tác với
nhau một cách dễ dàng, thông qua các quy định và hợp đồng được định nghĩa trước.

Cụ thể, khi chúng ta sử dụng một ứng dụng hoặc trang web, chúng ta đang tương
tác với một API nào đó. Khi chúng ta muốn xem một trang web, chúng ta sẽ gửi yêu
cầu đến máy chủ (server) thông qua API và nhận lại các dữ liệu và thông tin
tương ứng để hiển thị trang web đó. Tương tự, khi chúng ta muốn thực hiện một
thao tác nào đó trên trang web, chúng ta sẽ tương tác với API để gửi yêu cầu và
nhận lại các kết quả tương ứng.

API có thể được phát triển và cung cấp bởi một công ty hoặc một nhóm lập trình
viên, hoặc được cung cấp bởi các dịch vụ bên thứ ba như Google Maps API hay
Facebook API. API cũng có thể được sử dụng để cho phép các ứng dụng và dịch vụ
khác nhau kết nối và tương tác với nhau, tạo ra sự liên kết giữa các ứng dụng và
dịch vụ khác nhau trên nền tảng web.

Trong lĩnh vực phát triển web, API thường được sử dụng để cho phép các ứng dụng
web tương tác với các dữ liệu và thông tin từ các máy chủ khác nhau, giúp tạo ra
các ứng dụng web đa nền tảng và tích hợp với các dịch vụ khác nhau. Ngoài ra,
API cũng được sử dụng trong phát triển các ứng dụng di động và ứng dụng máy tính
để bàn để cho phép các ứng dụng này kết nối và tương tác với các dữ liệu và
thông tin từ các nguồn khác nhau.

Trên thị trường hiện nay, có rất nhiều loại API được phát triển và cung cấp, từ
các API mở để phát triển ứng dụng đa nền tảng cho đến các API dành riêng cho các
ứng dụng di động và các ứng dụng máy tính để bàn.
