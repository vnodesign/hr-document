# API là gì?

![API là gì](/static/docs/api.jpg)

API là giao thức giúp kết nối nhiều ứng dụng website với nhau khi lập trình backend.

Với API, lập trình viên có thể dễ dàng kết nối phần mềm, CSDL và dịch vụ của mình hoặc của người khác lại với nhau mà không cần đến những đoạn code phức tạp. Ngoài ra, data và ứng dụng website của lập trình viên cũng có giá trị hơn khi được người khác tận dụng khi phát triển sản phẩm.
