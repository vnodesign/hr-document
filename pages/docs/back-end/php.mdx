# Ngôn ngữ lập trình PHP là gì?

![Ngôn ngữ lập trình PHP là gì](/static/docs/php.png)

PHP là viết tắt của "Hypertext Preprocessor" là một ngôn ngữ lập trình mã nguồn mở được sử dụng chủ yếu để phát triển các ứng dụng web động và các trang web tĩnh. PHP được thiết kế để tích hợp dễ dàng vào HTML, và nó được thực thi trên máy chủ web để tạo nội dung động. PHP được phát triển ban đầu bởi Rasmus Lerdorf vào năm 1994 và hiện nay được phát triển và duy trì bởi một cộng đồng lớn các lập trình viên trên toàn thế giới.
