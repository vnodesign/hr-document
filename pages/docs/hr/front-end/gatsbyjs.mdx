# GatsbyJS là gì?

![GatsbyJS là gì](/static/docs/gatsbyjs.png)

GatsbyJS là một framework mã nguồn mở dựa trên ReactJS được sử dụng để xây dựng
các trang web tĩnh hoặc các trang web tĩnh với tính năng động. Nó cho phép các
nhà phát triển xây dựng các trang web tốt về mặt SEO, tốc độ và hiệu suất. Nó
cũng cung cấp các công cụ để tạo các ứng dụng web tĩnh tốt hơn và dễ quản lý
hơn.

GatsbyJS sử dụng các công nghệ như GraphQL, Webpack và ReactJS để tạo ra các
trang web tĩnh động. Nó cho phép các nhà phát triển lấy dữ liệu từ nhiều nguồn
khác nhau như CMS, API hoặc các tệp dữ liệu tĩnh và kết hợp chúng để tạo ra các
trang web tĩnh. Các trang web được tạo ra bởi GatsbyJS được tối ưu hóa cho tốc
độ và hiệu suất cao, vì vậy các trang web sẽ tải nhanh hơn và cung cấp trải
nghiệm người dùng tốt hơn.

GatsbyJS cũng cung cấp nhiều plugin và theme cho phép các nhà phát triển mở rộng
và tùy chỉnh các trang web của họ. Nó cũng cung cấp các tính năng như tạo trang
và tiền tạo trang (pre-rendering), tự động tối ưu hóa hình ảnh, và nhiều tính
năng khác giúp các nhà phát triển xây dựng các trang web tĩnh một cách dễ dàng
và hiệu quả.

Tóm lại, GatsbyJS là một framework mã nguồn mở được sử dụng để xây dựng các
trang web tĩnh hoặc các trang web tĩnh với tính năng động. Nó sử dụng các công
nghệ tiên tiến như GraphQL, Webpack và ReactJS để tạo ra các trang web tĩnh động
tối ưu hóa tốc độ và hiệu suất cao. Nó cũng cung cấp nhiều plugin và theme cho
phép các nhà phát triển tùy chỉnh các trang web của họ một cách dễ dàng.
