# NextJS là gì?

![NextJS là gì](/static/docs/nextjs.png)

NextJS là một framework mã nguồn mở của ReactJS được phát triển bởi Vercel. Nó
cho phép xây dựng các ứng dụng web phía client và server-rendered với hiệu suất
cao và khả năng tương tác tốt.

NextJS có các tính năng như pre-rendering, hot module replacement, automatic
code splitting và routing. Pre-rendering cho phép ứng dụng web được render trước
khi được truy cập bởi người dùng, cải thiện thời gian tải trang và trải nghiệm
người dùng. Hot module replacement giúp các nhà phát triển cập nhật mã của ứng
dụng mà không cần tải lại trang. Automatic code splitting giúp tách các thành
phần của ứng dụng thành các bundle nhỏ để tối ưu hóa tải trang. Routing được xây
dựng sẵn để quản lý các đường dẫn và thực hiện việc điều hướng trên trang.

NextJS cũng tích hợp sẵn các công nghệ như Webpack, Babel, ESLint và Sass để cải
thiện quá trình phát triển và tạo ra mã đáng tin cậy. Nó cũng hỗ trợ các tính
năng như CSS Modules, Serverless Functions và API Routes.

NextJS được sử dụng rộng rãi trong các ứng dụng web lớn, từ các trang thương mại
điện tử đến các trang web tin tức và giáo dục. Vercel cũng cung cấp một dịch vụ
hosting để giúp các nhà phát triển triển khai và quản lý các ứng dụng NextJS của
họ.
