# Front End là gì?

![Facebook Newfeed](/static/docs/facebook-newfeed.png)

Front End là quy trình thiết kế và xây dựng giao diện người dùng trên trang web
hoặc ứng dụng web sử dụng các ngôn ngữ HTML, CSS và JavaScript để tạo ra một
trải nghiệm tương tác cho người dùng. Chức năng của Front End là hiển thị nội
dung và cho phép người dùng tương tác với các phần của trang web hoặc ứng dụng
web.

Các công nghệ Front End phổ biến hiện nay bao gồm các Framework trong JavaScript
như ReactJS, VueJS, AngularJS, v.v. Các Framework này giúp các nhà phát triển
tăng tốc độ và giảm độ phức tạp khi xây dựng các ứng dụng web, đồng thời cung
cấp cho họ các tính năng và khả năng tùy chỉnh tốt hơn. Các công nghệ Front End
cũng bao gồm các thư viện và công cụ hỗ trợ khác như Bootstrap, Sass, jQuery,
v.v.

Ví dụ, khi bạn truy cập vào trang Facebook, các phần của trang web (như các nút,
bảng tin, hình ảnh, v.v.) mà bạn tương tác được là các phần của Front End. Các
phần này được tạo ra bằng cách sử dụng HTML để xây dựng cấu trúc, CSS để thiết
kế và định dạng, và JavaScript để cung cấp các tính năng tương tác.

Tóm lại, Front End là quy trình thiết kế và xây dựng giao diện người dùng trên
trang web hoặc ứng dụng web, và sử dụng các công nghệ như HTML, CSS và
JavaScript để tạo ra các phần của giao diện. Các công nghệ Front End bao gồm các
Framework, thư viện và công cụ hỗ trợ, giúp các nhà phát triển xây dựng các ứng
dụng web nhanh hơn và tùy chỉnh tốt hơn.

## HTML là gì?

HTML (Hypertext Markup Language) là ngôn ngữ đánh dấu siêu văn bản được sử dụng
để xây dựng các trang web. HTML cho phép người dùng định nghĩa cấu trúc và nội
dung của trang web bằng cách sử dụng các thẻ đánh dấu. Mỗi thẻ HTML được đặt
trong cặp dấu ngoặc nhọn và được sử dụng để định nghĩa các phần tử của trang
web, chẳng hạn như đầu trang (header), chân trang (footer), các đoạn văn bản
(text), hình ảnh (images), đường dẫn (links), bảng (tables), biểu đồ (charts),
các phần tử đa phương tiện như âm thanh và video (audio và video), và các biểu
mẫu nhập liệu (forms).

HTML được phát triển bởi Tim Berners-Lee và đội ngũ của ông tại CERN vào những
năm 1980, với mục đích tạo ra một hệ thống giao tiếp trực tuyến giữa các nhà
nghiên cứu trên toàn thế giới. Từ đó đến nay, HTML đã trở thành một tiêu chuẩn
công nghiệp cho phát triển web và được sử dụng rộng rãi trên toàn thế giới.

HTML được hỗ trợ bởi các trình duyệt web phổ biến như Google Chrome, Mozilla
Firefox, Apple Safari, Microsoft Edge và Opera. Các phiên bản HTML mới nhất bao
gồm HTML5, cung cấp cho nhà phát triển các tính năng đa phương tiện, thiết kế
đáp ứng và hỗ trợ di động tốt hơn.

## CSS là gì?

CSS (Cascading Style Sheets) là một ngôn ngữ được sử dụng để mô tả cách thức
hiển thị của các tài liệu HTML hoặc XML trên các trình duyệt web. Nó cho phép
các nhà phát triển web tạo ra các trang web có giao diện đẹp, dễ đọc và dễ sử
dụng hơn.

Với CSS, các nhà phát triển có thể tạo ra các kiểu định dạng, bố cục và trang
trí cho các phần tử HTML. CSS sử dụng các luật và thuộc tính để chỉ định các
phong cách cho các phần tử HTML như font chữ, màu sắc, kích thước, khoảng cách,
vị trí và hình dạng.

CSS được sử dụng để tách riêng phần kiểu định dạng của trang web từ phần nội
dung. Điều này cho phép các nhà phát triển web tạo ra các trang web linh hoạt và
dễ bảo trì hơn, bởi vì nó giúp họ thay đổi các phong cách của các phần tử trên
trang web một cách dễ dàng và hiệu quả.

Một trong những lợi ích lớn nhất của CSS là khả năng tạo ra các trang web tương
thích với nhiều loại trình duyệt khác nhau. Với CSS, các nhà phát triển web có
thể chỉ định các phong cách cho các phần tử HTML một lần duy nhất và chúng sẽ
được áp dụng cho tất cả các trình duyệt web. Điều này giúp giảm thiểu thời gian
và công sức cần thiết để phát triển các trang web và cải thiện trải nghiệm người
dùng trên nhiều thiết bị và trình duyệt khác nhau.

Tổng quan, CSS là một công cụ hữu ích và cần thiết để phát triển các trang web
đẹp, dễ sử dụng và tương thích với nhiều thiết bị và trình duyệt khác nhau.

## JavaScript là gì?

JavaScript là một ngôn ngữ lập trình thông dịch, đa nền tảng, dùng để phát triển
ứng dụng web và trình duyệt. Ban đầu, JavaScript được phát triển để tương tác
với người dùng trên trình duyệt, nhưng sau đó đã được sử dụng cho nhiều mục đích
khác nhau bao gồm phát triển ứng dụng di động, ứng dụng máy chủ và trò chơi.

JavaScript là ngôn ngữ lập trình client-side, nghĩa là nó được thực thi trên
trình duyệt của người dùng. Nó cho phép tương tác với HTML và CSS để tạo ra các
trang web tương tác động, đáp ứng và động.

Một trong những đặc điểm nổi bật của JavaScript là tính năng "hoạt động động"
(dynamic), cho phép các trang web thay đổi nội dung và hành vi mà không cần tải
lại trang. Điều này làm cho trang web trở nên thú vị hơn và cho phép người dùng
tương tác trực tiếp với trang web mà không cần gửi yêu cầu đến máy chủ.

JavaScript được phát triển bởi Netscape vào những năm 1990 và sau đó được chuẩn
hóa bởi ECMA International thông qua chuẩn ECMAScript. Hiện tại, JavaScript là
một trong những ngôn ngữ lập trình phổ biến nhất trên thế giới và được sử dụng
rộng rãi trong phát triển ứng dụng web, di động và máy chủ.
