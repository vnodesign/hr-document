# Front End là gì?

![Facebook Newfeed](/static/docs/facebook-newfeed.png)

Front End là quy trình thiết kế và xây dựng giao diện người dùng trên trang web hoặc ứng dụng web sử dụng các ngôn ngữ HTML, CSS và JavaScript để tạo ra một trải nghiệm tương tác cho người dùng. Chức năng của Front End là hiển thị nội dung và cho phép người dùng tương tác với các phần của trang web hoặc ứng dụng web.

Các công nghệ Front End phổ biến hiện nay bao gồm các Framework trong JavaScript như ReactJS, VueJS, AngularJS, v.v. Các Framework này giúp các nhà phát triển tăng tốc độ và giảm độ phức tạp khi xây dựng các ứng dụng web, đồng thời cung cấp cho họ các tính năng và khả năng tùy chỉnh tốt hơn. Các công nghệ Front End cũng bao gồm các thư viện và công cụ hỗ trợ khác như Bootstrap, Sass, jQuery, v.v.

Ví dụ, khi bạn truy cập vào trang Facebook, các phần của trang web (như các nút, bảng tin, hình ảnh, v.v.) mà bạn tương tác được là các phần của Front End. Các phần này được tạo ra bằng cách sử dụng HTML để xây dựng cấu trúc, CSS để thiết kế và định dạng, và JavaScript để cung cấp các tính năng tương tác.

Tóm lại, Front End là quy trình thiết kế và xây dựng giao diện người dùng trên trang web hoặc ứng dụng web, và sử dụng các công nghệ như HTML, CSS và JavaScript để tạo ra các phần của giao diện. Các công nghệ Front End bao gồm các Framework, thư viện và công cụ hỗ trợ, giúp các nhà phát triển xây dựng các ứng dụng web nhanh hơn và tùy chỉnh tốt hơn.
