# Bootstrap là gì?

![Bootstrap là gì](/static/docs/bootstrap.jpg)

Bootstrap là một framework HTML, CSS và JavaScript phổ biến được sử dụng để phát
triển các trang web đáp ứng (responsive). Với Bootstrap, lập trình viên có thể
dễ dàng xây dựng các thành phần cơ bản như typography, forms, buttons, tables,
grids, navigation và image carousels một cách nhanh chóng và tiện lợi.

Bootstrap được thiết kế để giảm thiểu thời gian và công sức cần thiết để xây
dựng các trang web, đồng thời tăng tính khả chuyển của các trang web bằng cách
tự động thích nghi với các kích thước màn hình khác nhau. Nó cũng cung cấp các
kiểu CSS chuẩn, giúp đảm bảo sự thống nhất và đồng nhất giữa các thành phần trên
trang web.

Bootstrap cũng hỗ trợ các tính năng JavaScript như tooltip, modal, scrollspy,
popover, carousel, v.v. để cung cấp các tính năng tương tác cho trang web.

Ngoài ra, Bootstrap còn có một cộng đồng lớn với nhiều tài liệu, ví dụ và hỗ
trợ, giúp cho việc học tập và sử dụng Bootstrap trở nên dễ dàng hơn đối với các
lập trình viên.

Tóm lại, Bootstrap là một framework HTML, CSS và JavaScript được sử dụng rộng
rãi để xây dựng các trang web đáp ứng và cung cấp các thành phần cơ bản, kiểu
CSS chuẩn và các tính năng JavaScript để giảm thiểu thời gian và công sức cần
thiết để xây dựng các trang web. Các tính năng này giúp lập trình viên tạo ra
các trang web đẹp mắt và đáp ứng nhanh chóng.
