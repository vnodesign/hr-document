# Nginx là gì?

![Nginx là gì](/static/docs/nginx.jpg)

Nginx là một phần mềm máy chủ web miễn phí và mã nguồn mở được phát triển để
cung cấp các dịch vụ máy chủ web HTTP và Reverse Proxy. Nó được phát triển ban
đầu để giải quyết các vấn đề hiệu suất của các máy chủ web khác như Apache.

Nginx có thể chạy trên nhiều nền tảng khác nhau, bao gồm các hệ điều hành
Windows, Linux, macOS và Unix. Nó được thiết kế để xử lý nhiều kết nối đồng thời
và cung cấp hiệu suất tốt hơn trong môi trường có tải lớn.

Ngoài việc cung cấp các dịch vụ máy chủ web, Nginx cũng có thể được sử dụng để
cân bằng tải, chạy ứng dụng web và các dịch vụ mạng khác. Nó cung cấp tính năng
bảo mật như hỗ trợ mã hóa SSL/TLS.

Nginx là một trong những phần mềm máy chủ web phổ biến nhất trên thế giới và
được sử dụng bởi hàng triệu trang web trên toàn thế giới. Nó cung cấp tính năng
mở rộng, tùy chỉnh và quản lý các trang web phức tạp, bao gồm các trang web
thương mại điện tử và các ứng dụng web phức tạp.
