# Server là gì?

![Server là gì](/static/docs/web-server.jpg)

Một server, còn được gọi là máy chủ, là một máy tính (bao gồm phần cứng và phần mềm) được kết nối với mạng máy tính hoặc Internet. Trên máy chủ, được cài đặt các phần mềm hoặc được trang bị với các máy tính chuyên dụng, hoặc nhiều máy tính được nối mạng với khả năng lưu trữ để cung cấp các dịch vụ và tài nguyên cho các máy tính khác truy cập.
