# OpenLiteSpeed là gì?

![OpenLiteSpeed là gì](/static/media/openlitespeed.png)

OpenLiteSpeed là phiên bản mã nguồn mở của LiteSpeed WebServer Enterprise. Nó sở hữu đầy đủ những tính năng cần thiết được tìm thấy trong LiteSpeed WebServer gồm LScache. Đây là phiên bản phục vụ cho website có traffic cao.
