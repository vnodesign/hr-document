# VueJS

Năm 2014, framework VueJS được ra mắt bởi một kĩ sư phần mềm làm việc tại Google. Vue là sự kết hợp giữa Angular và React, thích hợp cho việc tạo ra các giao diện tương thích và các trang ứng dụng.

Như các framework khác, Vue sử dụng các ngôn ngữ truyền thống như HTML, CSS và JavaScript. Framework có tiện ích CLI riêng biệt, cung cấp khả năng xây dựng hiệu quả hoặc thử nghiệm ý tưởng mới. Ngoài ra, Vue có thể thực hiện nhiều tác vụ cùng lúc. Nó có thể giải quyết tốt các quá trình phức tạp như tạo ứng dụng web hay ổn định các ứng dụng di động.
