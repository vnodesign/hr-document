# Hệ điều hành Linux là gì

- Nó là một hệ điều hành dành cho máy chủ.
- Nó có thể được cài trên các thiết bị máy tính hiện nay dành cho cá nhân.
- Nó là một hệ điều hành được sử dụng nhiều nhất bởi các lập trình viên.
- Có rất nhiều phiên bản dành cho hệ điều hành Linux này.
- Nó được sử dụng hầu hết bởi các công ty về IT, Software, v.v.

## Trong hệ điều hành Linux, thì có những phiên bản nào?

- Hệ điều hành Ubuntu
- Hệ điều hành Fedora (Red hat)
- Hệ điều hành Linux Mint
- Hệ điều hành Kali Linux
- Hệ điều hành CentOS
- Hệ điều hành Android
