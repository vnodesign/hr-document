# Bootstrap là gì?

![Bootstrap là gì](/static/media/bootstrap.jpg)

**Bootstrap** là một **framework** HTML, CSS, và JavaScript phổ biến nhất để phát triển website chuẩn responsive.

**Bootstrap** giúp cho quá trình thiết kế website của lập trình viên diễn ra nhanh chóng dựa trên những thành tố cơ bản sẵn có như typography, forms, buttons, tables, grids, navigation, image carousels,…
