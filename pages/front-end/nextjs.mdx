# NextJS là gì?

![NextJS là gì](/static/media/nextjs.png)

**NextJS là một framework được xây dựng từ React**, Node, Webpack và Babel. Nó được tạo ra để giúp các lập trình viên tạo React App có tính năng SSR (Server Side Render) hoặc web app tĩnh.

NextJS được đóng góp bởi Vercel. NextJS cung cấp giải pháp lập trình web sử dụng một hosting bao trọn cả Back End lẫn Front End.
