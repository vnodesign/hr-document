# GatsbyJS là gì?

![GatsbyJS là gì](/static/media/gatsbyjs.png)

**GatsbyJS là một framework mã nguồn mở và miễn phí được build dựa trên React**. GatsbyJS giúp lập trình viên xây dựng websites và apps với tốc độ cực nhanh

Gatsby là một công cụ tạo static site (trang web tĩnh) với dữ liệu được lấy từ bất kỳ nguồn nào và hiển thị chúng dưới dạng HTML tĩnh.

Các trang web sử dụng Gatsby giúp lập trình viên tạo ra các dynamic web apps từ blog đến các trang e-commerce..., lập trình viên sẽ viết code như React rồi khi build site nó sẽ tạo ra các file HTML, CSS và JS. Sau đó lập trình viên có thể tải lên các CMS như Contentful, Drupal, WordPress.

Gatsby lấy dữ liệu từ bất kỳ nguồn dữ liệu nào, ví dụ như Markdown, CMS như Contentful, WordPress,... Nó sử dụng GraphQL để lấy dữ liệu theo ý muốn.
