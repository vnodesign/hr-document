# Mithril là gì?

![Mithril là gì](/static/media/mithril.jpg)

**Mithril là một JavaScript framework gọn nhẹ nhưng rất mạnh mẽ, được dùng để xây dựng các ứng dụng web một trang (Single Page Applications - SPA)**. Mithril được viết hoàn toàn bằng ES5, hỗ trợ xây dựng giao diện với virtual DOM, cung cấp một hệ thống routing và một số tiện ích khác như XHR, stream...Tất cả gói gọn trong một tập tin JS nhỏ hơn 8KB gzip. Giao diện viết bằng mithril có thể hoạt động trên tất cả trình duyệt từ IE9 mà không cần bất cứ polyfill nào.
