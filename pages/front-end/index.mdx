# Front End là gì

![Facebook Newfeed](/facebook-newfeed.png)

**Front End là cách gọi quy trình sử dụng các ngôn ngữ HTML, CSS và JavaScript**, khi thiết kế và xây dựng giao diện cho các trang web hoặc ứng dụng web để người dùng có thể xem và tương tác trực tiếp trên đó (lấy Facebook làm ví dụ, thì khi bạn đọc bài viết bất kỳ trên newfeed, thì chính nội dung mà bạn nhìn thấy nó là Front End).

Các công nghệ Front End thường bao gồm các Framework trong JavaScript như: **ReactJS**, **VueJS**, **NuxtJS**, **AngularJS**, v.v.
